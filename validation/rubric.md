## Using This Rubric

Load this file when validating a new skill. Score each criterion on a 1-5 scale:

- **5** — Excellent; exemplary practice demonstrated clearly
- **4** — Good; meets expectations with minor gaps
- **3** — Adequate; acceptable but could be improved
- **2** — Weak; significant gaps or inconsistencies
- **1** — Poor; major deficiencies

Average scores across all criteria to determine overall skill quality.
