# Rovo Docs Site Navigation

## Rovo Overview

Rovo is Atlassian's cloud-native AI assistant for the modern workplace. Rovo helps teams search across their
work, understand connections, and automate tasks through natural language interaction. Rovo includes Search, Chat,
Agents, and a CLI (Rovo Dev) for advanced automation and IDE integration.

**Deployment**: Cloud-only (Atlassian Cloud)

**Availability**: Not available in Government Cloud

**Related Products**: Part of Atlassian AI family alongside Compass and Loom

## Rovo Feature Areas

### Core Features (`/using-rovo/`)

- **Search** — Search across Jira, Confluence, and connected data sources
- **Chat** — AI chat interface with reasoning and prompting modes
- **Agents** — Create AI agents for automation tasks
- **Rovo Button** — Contextual AI assistance in Jira and Confluence
- **Definitions** — Semantic understanding of work items and concepts
- **Browser Extension** — Rovo access from anywhere on the web

### Rovo Dev (`/work-with-rovo-dev/`)

- **CLI** — Command-line interface for automation and scripting
- **Sessions** — Stateful agent sessions with memory
- **MCP Servers** — Model Context Protocol integration
- **Code Reviews** — AI-powered code review automation
- **IDE Integration** — VS Code, Cursor, JetBrains support
- **Bitbucket Pipelines** — CI/CD integration
- **Jira Integration** — Issue tracking and workflow automation
- **Automations Builder** — Visual automation creation

### Organization Administration (`/manage-rovo/`)

- **Admin Console** — Org-level Rovo management
- **Connector Management** — Connect data sources (Jira, Confluence, etc.)
- **User Permissions** — Role-based access control
- **Data Privacy** — Data handling and compliance
- **Quotas & Limits** — Usage monitoring and administration
- **Billing** — Subscription and license management

## URL Pattern Convention

Rovo docs follow this pattern:

```text
https://support.atlassian.com/rovo/docs/[topic-slug]/
```

Example: `https://support.atlassian.com/rovo/docs/rovo-search/`

Rovo Dev CLI docs:

```text
https://support.atlassian.com/rovo/docs/work-with-rovo-dev/[command-topic]/
```

Example: `https://support.atlassian.com/rovo/docs/work-with-rovo-dev/rovo-dev-cli-reference/`

## Cloud & Availability Notes

- **Cloud-only**: Rovo requires Atlassian Cloud. Server and Data Center are not supported.
- **Government Cloud**: Not available in Atlassian Government Cloud.
- **Data residency**: Data is processed in your Atlassian Cloud region.
- **AI privacy**: Review data privacy docs for AI processing details.
