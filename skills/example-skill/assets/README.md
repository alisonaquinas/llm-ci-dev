# Asset Templates

This directory contains template files and configuration examples for common CI/CD tasks.

## Available Templates

- `config-template.yml` — Example configuration file
- `pipeline-template.yaml` — Example pipeline definition
- `dockerfile-template` — Example Dockerfile

## Usage

Copy and adapt these templates for your use case. Do NOT load these files into the context
when using skills — they are for reference only. Extract relevant content and adapt it
for your specific scenario.

## Template Development

When adding new templates:
1. Use descriptive filenames ending with `-template`
2. Include comments explaining key sections
3. Provide realistic but safe example values
4. Document any required customization
