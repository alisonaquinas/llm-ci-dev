# Installation and Setup

This reference provides installation and configuration guidance for the example skill.

## Prerequisites

- MacOS, Linux, or Windows with WSL2
- Bash shell (or PowerShell equivalent on Windows)

## Installation

### Step 1: Install Prerequisites

```bash
# Example command
apt-get install example-tool
```

### Step 2: Configure

```bash
# Configuration step
example-tool --init
```

### Step 3: Verify

```bash
# Verification step
example-tool --version
```
