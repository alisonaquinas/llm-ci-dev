# Quick Reference

Common commands and syntax for day-to-day use.

## Basic Commands

```bash
# List resources
example-command list

# Get details
example-command detail <id>

# Create new resource
example-command create --name "resource-name"

# Delete resource
example-command delete <id>
```

## Common Patterns

### Pattern 1: Working with Configurations

```bash
example-command config set key=value
example-command config get key
```

### Pattern 2: Debugging

```bash
example-command --verbose <subcommand>
example-command debug --level=5
```
