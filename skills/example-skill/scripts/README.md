# Example Skill Scripts

This directory contains example scripts demonstrating common automation tasks.

## Available Scripts

- `setup.sh` — Initial setup and configuration
- `deploy.sh` — Deploy workflow
- `cleanup.sh` — Cleanup resources

## Usage

Run scripts directly from the terminal:

```bash
bash scripts/setup.sh
bash scripts/deploy.sh
```

## Script Development Notes

- All scripts should include error handling
- Use meaningful variable names
- Add comments explaining complex logic
- Test thoroughly before committing
