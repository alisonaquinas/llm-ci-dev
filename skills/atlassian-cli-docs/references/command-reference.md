# Atlassian CLI Command Reference

Complete command tree for the Atlassian CLI organized by product and function.

## Admin Commands

| Command | Function | URL |
| --- | --- | --- |
| `admin auth login` | Authenticate as admin | `https://developer.atlassian.com/cloud/acli/reference/admin/auth/login/` |
| `admin auth logout` | Logout admin session | `https://developer.atlassian.com/cloud/acli/reference/admin/auth/logout/` |
| `admin auth whoami` | Show current admin user | `https://developer.atlassian.com/cloud/acli/reference/admin/auth/whoami/` |
| `admin org list` | List organizations | `https://developer.atlassian.com/cloud/acli/reference/admin/org/` |
| `admin org info` | Get organization info | `https://developer.atlassian.com/cloud/acli/reference/admin/org/` |
| `admin feedback` | Submit feedback | `https://developer.atlassian.com/cloud/acli/reference/admin/feedback/` |
